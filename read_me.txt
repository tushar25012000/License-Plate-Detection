'ALPR_All.py' file is for ALPR GUI Recogniton. [For all country]

* Steps of installation

- Python 3.6.8 or 3.7 is working fine.
- Install tesseract OCR, the exe file which I have given in folder
- Install other requirements using 'pip install -r requirements.txt'
